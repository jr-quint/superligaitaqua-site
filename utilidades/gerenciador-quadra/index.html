            Instruções de Uso
            Times
            • Use os botões e na lista da fila para mover jogadores para os times Vermelho ou Azul.
            • Para retirar um jogador do time e voltar para a fila, use o botão ao lado do nome no time.
            • O jogador removido do time retornará ao final da fila.
            • Registre a derrota de um time clicando no botão no cabeçalho do time correspondente.
            Próximo Time
            • Adicione jogadores da fila ao Próximo Time usando o botão .
            • Obs: O próximo time será movido automaticamente para um dos lados da quadra ao registrar derrota de um dos times.
            • O botão no cabeçalho do Próximo Time também permite mover todos os jogadores para a quadra de uma vez.
            • Use o botão ao lado do nome de cada jogador para mover um jogador específico para a quadra.
            Time em Descanso
            • Times que alcançarem o número configurado de vitórias podem ser automaticamente enviados para descanso, se essa opção estiver ativada.
            • O time em descanso volta automaticamente para a quadra após derrota de um dos times.
            Fila
            • Jogadores recém adicionados entram no final da fila.
            • Use os botões ao lado do nome para mover jogadores para times ou próximo time, ou removê-los da fila.
            Configurações e Ações
            • Ajuste o número de jogadores por time, vitórias para descanso e habilite ou desabilite o descanso.
            • Use o botão "Resetar Tudo" para apagar todos os dados e começar do zero.
